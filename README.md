# CLI-Quiz-Game

## About
A simple quiz game that will ask you 5 questions and check the answers given and will return a score based on many were answered correctly.  

## How to run
- clone or download the repo
- do "npm install" within the terminal in order to install all necessary packages.
- run  "node main.js" to start the game.

## What I learned
- Learned to how to work and use objects to store data.
- Learned how to loop through objects in order to ask questions and check/compare answers.
